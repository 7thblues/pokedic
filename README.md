# Ruby/Sapphire Notes

Pokemon Ruby/Sapphire related notes and reference material.

## Files

- `story-flow-ko.md`: Main story event flow from new game to credits, based on `pokeruby` scripts.
- `pokeruby-analysis-notes-ko.md`: Source-analysis memory for future sessions, including data locations, battle logic pointers, and guide-generation notes.
- `ruby-sapphire-encounters.html`: Ruby/Sapphire version-exclusive encounters and higher encounter odds, based on `pokeruby` wild encounter data.
- `hoenn-pokedex-checklist.html`: Hoenn Pokedex 1-202 collection checklist with type, version, and obtain-method notes.
- `elite-four-party-builder.html`: Elite Four and Champion counter-party guide based on `pokeruby` trainer parties and movesets.
